Edumater: How does it work?


Overview

Edumater is designed to ask questions based on an intelligent algorithm the user can modify to fit their needs. What makes Edumater special is that it's designed to keep asking you the same questions over and over again over an extended period time (i.e., weeks or months) to ingrain the information into the user's brain.

Step One: Questions

The entire process starts with the questions. Questions can be added to your database at any point an time with the help of the Question Manager. Questions are organized into Nodes, which are very similar to the files and folders in an everyday computer's hard drive. Nodes can be added, removed and dragged around. And, more imporantly, every node can have questions attached to it with the "Add Question" button.

These nodes stick around until you close Edumater's tab. So: Save your stuff, and save it often, or you'll lose your questions. You have three options to save with different organization patterns. "Monolith" dumps everything into a single hard-to-edit file. The other methods allow you to add and remove nodes from outside Edumater.

Step Two: Scoresheets

Scoresheets define the algorithm which selects questions. Based on a number of criteria (including the numbers of times every question has been answered correctly, incorrectly, the time since it was last answered, and a user-specified Priority value), Edumater will choose the highest-value question every time a user needs to ask another question. This selection is based on the Current Scoresheet, which can be changed, tuned, uploaded, and saved from the Scoresheet Manager. At the bottom of the page is a particularly useful link, the Scoresheet Tuner, which lets you choose an example question and fine-tune the scoresheet around it in real-time.

Step Three: Ask Away!

With questions loaded and a scoresheet setup (the default scoresheet will usually do), you can head to the Questioner and start learning. As you do, Edumater will keep track of what questions you've gotten right and wrong, and decide on the best question to ask you from that. If you ever happenn to come up with some new fact to add to your database, you can always go back the Question Manager and add it.
